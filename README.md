#### TECHNOLOGIES


* AngularJs 
* GSAP - TweenMax
* Babel | ES2015
* Browserify
* JSHint
* Gulp
* SASS

Install dependancies :  
```shell
  npm i
```

Launch the project :  
```shell
  npm start
```

The project will be launched at http://localhost:3000/

Build for production :  
```shell
  npm run build
```
